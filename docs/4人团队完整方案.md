## 三、新增功能详细设计

### 3.1 陷阱系统（A负责）

```cpp
enum class TrapType {
    SPIKE,      // 地刺 -5HP
    POISON,     // 毒雾，中毒3回合
    TELEPORT    // 传送到随机房间
};

class Trap {
    TrapType type_;
    int x_, y_;
    bool triggered_;
public:
    void trigger(Player& player);
};
```

地图生成时，每层随机放置 3-5 个陷阱，用 `^` 表示（踩上去才触发）。

### 3.2 门与宝箱系统（A负责）

```cpp
class Door {
    bool locked_;
    int x_, y_;
public:
    bool tryOpen(Player& player);  // 需要钥匙
};

class Chest {
    std::vector<Item*> contents_;
    bool isTrap_;  // 20%概率是陷阱宝箱
    bool opened_;
public:
    void open(Player& player);
};
```

- 门用 `+` 表示，锁门用 `+`（红色）表示
- 宝箱用 `□` 表示

### 3.3 音效系统（C负责）

Windows 用 `Beep(频率, 时长)` 播放简单音效：

```cpp
class Sound {
public:
    static void playAttack();     // Beep(800, 100)
    static void playHit();        // Beep(400, 150)
    static void playLevelUp();    // 连续三声 Beep
    static void playVictory();    // 胜利旋律
    static void playGameOver();   // 失败音效
};
```

Linux 用 `\a` 响铃（简化版）。

### 3.4 成就系统（D负责）

```cpp
enum class AchievementID {
    FIRST_KILL,
    RICH_MAN,
    COMBO_MASTER,
    // ... 共10个
};

class Achievement {
    AchievementID id_;
    std::string name_;
    std::string desc_;
    bool unlocked_;
public:
    void unlock();
};

class AchievementManager {
    std::vector<Achievement> achievements_;
public:
    void check(Player& player);  // 每回合检查
    void display();              // 显示成就列表
};
```

### 3.5 商店系统（D负责）

每层进入前显示商店界面，玩家可用金币购买道具。

```cpp
struct ShopItem {
    std::string name;
    int price;
    ItemType type;
    int value;
};

class Shop {
    std::vector<ShopItem> items_;
public:
    void open(Player& player);  // 显示商店界面
    bool buyItem(Player& player, int index);
};
```
